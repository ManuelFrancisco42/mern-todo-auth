# MERN Auth Template

## Features
- Express backend
- MongoDB connection
- JWT Authentication
- Register/Login routes
- Vite React frontend

## Setup

### 1. Clone from template
Click "Use this template"

### 2. Install dependencies

Backend:
cd backend
npm install

Frontend:
cd frontend
npm install

### 3. Setup environment

Copy:
.env.example → .env

Fill values

### 4. Run project

Backend:
npm run dev

Frontend:
npm run dev
